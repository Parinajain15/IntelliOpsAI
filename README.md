# IntelliOps AI

AI-Powered Workforce Intelligence Dashboard built using ASP.NET Core, C#, SQL Server, and Gemini AI.

## Features
- Employee work log management
- Productivity analytics dashboard
- AI-generated operational summaries
- Smart workflow insights
- Enterprise-style reporting system

## Tech Stack
- ASP.NET Core MVC
- C#
- SQL Server
- Bootstrap
- Gemini API

## Project Goal
To develop an AI-assisted enterprise workflow intelligence platform inspired by modern operational management systems.
